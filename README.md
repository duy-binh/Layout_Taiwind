# Building Layout Taiwind

## Demo

https://github.com/duy-binh/Layout_Taiwind.git

## Run Project Here

Clone the project

```bash
  git clone https://github.com/duy-binh/Layout_Taiwind.git
```

Start the server

```bash
   npx tailwindcss -i ./src/CSS/input.css -o ./src/CSS/output.css -- watch
```
